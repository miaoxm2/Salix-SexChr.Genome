## 01_genome_assembly_annotation

### **Assembly/**

De novo draft genome assembly using HiFi long reads and scaffolding using HiC contact map. 
- HiFiasm v0.15.4
- purge_dups v1.2.5
- Juicer v1.6
- 3D-DNA

### **Annotation/**

Repeat prediction, gene structure prediction and gene function annotation.
- RepeatMasker v4.1.5
- RepeatModler v2.0.3
- LTR_retriever
- trf v4.10.0
- BRAKER
- TSEBRA
- EGGNOG-MAPPER
- InterProScan
- BLAST
